# insertion-sort-analysis
Empirical Analysis of Insertion Sort
====================================

Theoretical Complexities:
Best Case    : Θ(n)
Average Case : Θ(n^2)
Worst Case   : Θ(n^2)

Input(n)   Best Time(s)    Average Time(s) Worst Time(s)  
1000       0.000217        0.036634        0.064988       
2000       0.000386        0.161666        0.286208       
4000       0.000785        0.640475        1.215569       
8000       0.001620        2.574668        5.016871       

Observation:
Best case grows linearly.
Average and worst cases grow quadratically.
